# CodeAlpha_ImageGallery

A sleek, responsive, and interactive Image Gallery built as part of my Frontend Development internship at **CodeAlpha**. This project showcases the ability to create a clean user interface, handle image layouts dynamically, and implement interactive features like a lightbox view.

## 🚀 Features
* **Responsive Grid Layout:** Looks great on desktops, tablets, and mobile screens.
* **Interactive Lightbox:** Click on any image to open it in a full-screen modal view with next/previous controls.
* **Smooth Transitions:** Elegant hover effects and fade-ins for a modern user experience.

## 🛠️ Technologies Used
* **HTML5:** For semantic web structure.
* **CSS3:** For styling, grid layout, and smooth animations.
* **JavaScript (ES6):** For modal functionality, dynamic image loading, and interactive elements.
